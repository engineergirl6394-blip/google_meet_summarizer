# Google Meet Summarizer

## Overview
This project automatically summarizes Google Meet conversations using AI/NLP techniques.

## Features
- Converts speech/text into summary
- Saves time in meetings
- Easy to use interface

## Tech Stack
- Python
- NLP
- APIs

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py
